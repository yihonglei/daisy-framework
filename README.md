# 前言

SpringBoot，Dubbo，SpringCloud 脚手架实战项目。

daisy-component：基础组件

daisy-basic: 基础服务

daisy-springboot：springboot 基本应用

daisy-springcloud：springcloud 基本应用

daisy-dubbo：dubbo 基本应用

daisy-manager：前后端分离权限管理系统
